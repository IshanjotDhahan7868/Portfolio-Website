WealthPilot

WealthPilot is a simple investing projection tool that helps everyday people understand the long‑term impact of their investment decisions. Users can model different contribution plans, expected returns and time horizons to see how their wealth could grow over time.

Problem

Many investment calculators are either overly simplistic or overly complex. They don’t let users compare different contribution packages, factor in inflation, or visualize how small changes in monthly savings affect long‑term wealth. As a result, people struggle to plan effectively for retirement or big purchases.

Solution

WealthPilot allows users to define multiple “packages” representing monthly contributions and expected annual returns. The tool calculates future values using compound interest formulas and displays interactive charts of the projected growth. Users can add inflation adjustments and see side‑by‑side comparisons of their packages.

Tech Stack

  • Vanilla JavaScript for calculations
  • Chart.js for interactive charts
  • Lightweight HTML/CSS without any frameworks

My Role & Challenges

I built WealthPilot as an exercise in creating a straightforward yet flexible financial calculator. I implemented compound interest math functions, built dynamic forms for users to add or remove packages, and integrated Chart.js for data visualization. The main challenge was structuring the UI to remain simple while supporting multiple scenarios and ensuring the math remained precise across different time periods.
